## ğŸ“‹ What is a Profile?

Profiles are JSON configuration files that pre-configure SAP Pro Toolkit with:

- **Quick Actions**: Deep links to common SAP workflows
- **Shortcuts**: Frequently accessed URLs and documentation
- **Environments**: Pre-configured SAP system templates (Production, Preview, Sales Demo)
- **AI Prompts**: Joule copilot prompt templates for various tasks
- **Notes**: Reference documentation, code snippets, and SAP Note collections

**Profile Schema:**
```json
{
  "version": "2.0",
  "profileName": "My Custom Profile",
  "profileDescription": "Description of what this profile contains",
  "profileIcon": "ai",
  "globalShortcuts": [...],
  "environments": [...],
  "notes": [...]
}
```
