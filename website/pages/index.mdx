## üöÄ Get Started

### Installation

1. **Download from Chrome Web Store** (coming soon)
2. **Or install manually:**
   - Download latest release from [GitHub](https://github.com/sidbhat/sf-pro-toolkit/releases)
   - Open `chrome://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked" and select the extension folder

### Quick Setup

1. **Choose a Profile**: Start with a pre-built profile or create your own
2. **Add Environments**: Configure your SAP tenants (Production, Preview, etc.)
3. **Customize Shortcuts**: Add your most-used URLs and deep links
4. **Try AI Features**: Test the AI search and prompt templates
